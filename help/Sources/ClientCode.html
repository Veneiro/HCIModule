Client Code

What is my client code

When you convert into a client of the supermarket you get a card where your code is written. The code is formed by 3 letters (The first letter of your name and surnames) and 3 numbers (Random numbers). So if you are called, for example, Pedro Rodríguez Fernández, you will get something like PRF152.

¿How I validate my number?

You don't have to do nothing, just introduce the number on the text area and press continue. If your code is valid you will procced to the game
